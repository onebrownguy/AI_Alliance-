# Austin AI Alliance Website

A modern, responsive website for the Austin AI Alliance (austin-ai.org) built with Next.js 16, TypeScript, and Tailwind CSS.

## Getting Started

### Prerequisites
- Node.js 18.17 or later
- npm

### Installation
1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

### Development
Run the development server:
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Build
To create an optimized production build:
```bash
npm run build
```

## Tech Stack
- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Deployment**: [Vercel](https://vercel.com)

## Project Structure
- `app/`: Application pages and layout
- `components/`: Reusable UI components
- `data/`: Content data files (Hardcoded for now)
- `public/`: Static assets

## Deployment
This project is ready to be deployed to Vercel. Connect your repository to Vercel or use the Vercel CLI:
```bash
vercel
```
